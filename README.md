# 🐍 Мои решения задач на Python

Здесь храняться задачи которые я решаю для общей практики в свободное время.

[![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🎯 О проекте

* **Цель:** Хранить и систематизировать мой опыт решения задач на Python.
* **Для кого:** Для потенциальных работодателей, которые хотят увидеть мой код, и для меня самого (чтобы видеть прогресс).
* **Стек:** Чистый Python, иногда с использованием `pytest` для тестов.

## 🗂️ Структура

* `basics/`: Фундаментальные задачи (циклы, условия, функции).
* `algorithms/`: Реализация алгоритмов (сортировки, поиск).
* `data_structures/`: Работа со структурами данных (списки, словари, множества, стеки).

## 🚀 Как начать
